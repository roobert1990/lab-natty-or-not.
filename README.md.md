## 📒 Descrição
Bem-vindo ao fascinante mundo da viagem no tempo! Neste projeto veremos como a IA generativa pode ser utilizada para simular e expandir nossas compreensões sobre este tema tão intrigante.

Usei o ChatGPT para criar uma introdução sobre o tema, consegui informações bem interessantes. Usei também o Visual Studio Code para editar o README e por ultimo a ASK AI para criar a imagem.
